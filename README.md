Project Description

This project is a responsive web interface built using HTML, CSS, and JavaScript. The layout is structured with Flexbox, ensuring adaptability across different screen sizes and devices.
To enhance the user experience, the project integrates the Swiper library, providing smooth and customizable sliders/carousels.

Key Features

Clean and semantic HTML structure

Responsive layout powered by CSS Flexbox

Interactive elements implemented with vanilla JavaScript

Image/Content sliders created using the Swiper library

Cross-browser compatibility and user-friendly design

Technologies Used

HTML5

CSS3 (Flexbox)

JavaScript

Swiper.js
